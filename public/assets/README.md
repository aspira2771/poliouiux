# Image assets

These power the look of the site. The real artwork is already committed here.
To swap any of them, just replace the file with the same name.

| File         | Where it's used                                   |
| ------------ | ------------------------------------------------- |
| `tree.jpg`   | Background of the hero (top) and ant colony (base)|
| `title.png`  | The "JiHyun's Life" plush title on the hero       |
| `ant.png`    | The ant — walking line, greeting, and celebrating |
| `mascot.png` | Extra character (not used by default)             |

Notes:
- `title.png` and `ant.png` have transparent backgrounds so they sit cleanly
  on the photo. If you replace them, export as PNG with transparency.
- If any file is missing, the site falls back to a hand-drawn SVG forest /
  balloon title, so it never breaks.
