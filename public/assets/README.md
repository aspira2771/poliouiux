# Background images

Drop your own background images here and they will be picked up automatically.
Until then, the site shows a soft forest-green gradient fallback.

Expected filenames (any of these can be omitted — each falls back to the gradient):

| File             | Where it's used                                  | Suggested size |
| ---------------- | ------------------------------------------------ | -------------- |
| `hero-tree.jpg`  | Hero screen behind the "JiHyun's Life" title     | 1920×1080+     |
| `tree-base.jpg`  | Ant colony scene (tree trunk + grass)            | 1920×1080+     |
| `forest-bg.jpg`  | Reserved for future scenes                       | 1920×1080+     |

Tip: landscape, high-resolution images look best. JPGs keep the build small.
